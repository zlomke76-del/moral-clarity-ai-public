# Public Release Notes

This repository is a public-facing Moral Clarity AI release.

It is designed to communicate product positioning, governance architecture, and public documentation without exposing private production infrastructure.

## Removed from public scope

- auth and admin routes
- founder login flows
- Stripe and billing workflows
- Supabase service-role logic
- customer data handling
- memory infrastructure
- newsroom workers
- support hooks
- debug and diagnostics routes
- operational telemetry
- internal orchestration modules

## Public scope

- landing page
- public product overview
- public governance framing
- public architecture boundary
- docs and contribution policy
