# Contributing

Thank you for your interest in Moral Clarity AI.

## Public Contributions

Public contributions are welcome for:

- documentation improvements
- copy clarity
- public examples
- accessibility improvements
- public-facing UI refinements

## Pull Requests

Please keep pull requests focused and minimal. Include a clear description of what changed and why.

## Boundary

This repository is public-facing only. Please do not submit production authentication flows, admin systems, billing systems, customer data, private telemetry, secrets, or internal orchestration logic.

## Security

Please do not disclose vulnerabilities publicly. See `SECURITY.md`.
