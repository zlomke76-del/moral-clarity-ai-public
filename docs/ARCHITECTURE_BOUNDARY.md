# Architecture Boundary

This public repository explains the Moral Clarity AI execution-governance ecosystem without exposing private production mechanics.

## Public layer

The public layer may include:

- product overview
- docs
- public examples
- sanitized demos
- conceptual architecture
- integration philosophy

## Private layer

The private layer should contain:

- authentication
- authorization
- admin workflows
- billing
- customer data
- memory systems
- model routing
- private telemetry
- enterprise orchestration
- runtime enforcement infrastructure

## Principle

Public repos explain the boundary. Private repos operate the boundary.
